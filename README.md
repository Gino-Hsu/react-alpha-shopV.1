## 介紹

以 React 進行開發，並以 CSS Modules 進行 CSS 樣式撰寫。
此專案為購物車，目前版本僅以 React 將畫面 UI 元素元件化，未來將加入狀態控制等功能。

完成的元件包含：
- Header
- Main (Register & Cart)
- Footer

### 使用指南

1. 將專案 clone 至本地，至終端機輸入：
   ```
   git clone https://github.com/Gino-Hsu/restaurant_list_CRUD.git
   ```
 
2. 執行專案，輸入：
   ```
   npm start
   ```
   
3. 瀏覽器 "http://localhost:3000" ，可瀏覽此專案。

4. 停止使用，至終端機：
   ```
   ctrl + c
   ```
